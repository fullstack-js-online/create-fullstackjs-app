# create-fullstackjs-app :penguin:

> Simple cli tool for generating fullstack-js-online starter projects

## Usage

```bash
yarn global @fullstackjs/create-fullstackjs-app
# or
npm i -g @fullstackjs/create-fullstackjs-app
```
